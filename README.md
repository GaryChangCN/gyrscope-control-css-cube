# gyrscope-control-css-cube

使用手机陀螺仪来控制css长方体

## 使用

```
git clone https://github.com/GaryChangCN/gyrscope-control-css-cube.git
npm update
npm start
//之后修改客户端和控制端 websocket 协议的地址为你服务端所在的局域网地址

```
